Sobre a Plataforma

Esta plataforma foi desenvolvida para estruturar o ensino musical, integrando o estudo prático em todas as 12 tonalidades da escala cromática, o acompanhamento teórico e o controle individual do progresso de cada aluno.

Todos os dados (lotes padrão e cadastros de alunos) são salvos de forma descentralizada diretamente no repositório do GitHub, garantindo acesso em qualquer dispositivo.

Painel Admin

Gestão de Lotes de Estudo

Área destinada à criação e edição de conjuntos de exercícios, tópicos ou músicas que podem ser importados rapidamente para os perfis dos alunos.

Como usar a área Admin:

  1. Token do GitHub: Cole e confirme seu Token de Acesso Pessoal (PAT) do GitHub no painel superior. Ele ficará salvo com segurança em seu navegador.
  2. Carregar e Editar Lotes no Editor:
    • Digite o ID do lote (Ex: modulo1, iniciante_pop) e clique em Carregar Lote.
    • O conteúdo do arquivo será exibido diretamente no textarea de edição.
    • Edite ou adicione novos itens escrevendo um item por linha. Se for de músicas, digite apenas o nome das canções.
    • Clique em Salvar / Sobrescrever Lote no GitHub para atualizar o arquivo no repositório.
  3. Cadastrar Novo Lote: Para criar um novo conjunto do zero, escolha o tipo de lote (Teoria, Tonalidades ou Músicas), informe um ID único (Ex: campo_harmonico), digite a lista no textarea e salve.
Alunos

Acompanhamento Individual do Aluno

É a sua planilha interativa para registrar o domínio de cada aluno nas 12 tonalidades, tópicos teóricos e músicas do repertório.

Passo a Passo de Uso:

  1. Buscar ou Criar Aluno: Digite o nome do aluno no campo de busca e clique em "Carregar / Criar". O sistema buscará o arquivo alunos/nome_aluno.json no GitHub ou iniciará um novo perfil.
  2. Importação em Lote: Para não precisar cadastrar item por item manualmente, digite o nome de um lote criado no Admin (Ex: modulo1) na aba correspondente e clique em Importar Lote. Os itens serão inseridos automaticamente na ficha do aluno.
  3. Marcação de Tonalidades e Tópicos: Conforme o aluno executar um exercício ou música em cada tom, clique sobre a caixa da tonalidade para marcar como concluído (de C a B).
  4. Salvar Progresso: Ao final da aula, clique no botão 💾 Salvar Progresso no GitHub para sincronizar as edições do aluno na nuvem.
💡 Dica Pedagógica: Incentive o aluno a buscar a conclusão das 12 tonalidades para cada música ou exercício cadastrado. Isso consolida a transposição automática e o domínio do instrumento.
Módulos de Prática

Tonalidades, Músicas e Teoria

Telas dedicadas ao treino dinâmico durante a aula:

  • Tonalidades: Prática focada em exercícios escalares, arpejos e padrões harmônicos transpostos.
  • Músicas: Visualização e prática do repertório trabalhado.
  • Conceitos: Registro do entendimento de conceitos teóricos fundamentais.
